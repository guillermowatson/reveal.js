Archivos

  • Objetos de tipo file
    • r: lectura (read).
    • w: escritura (write).
    • a: agregar (append).
    • b: binario (binary, Win).
    • +: lectura y escritura simultánea.
  • I/O Files

#!/usr/bin/python
# coding: utf-8

with open('archivo.txt', 'r+') as f:
    data = f.read()

Lectura de Archivos

  • read(): devuelve una cadena con el contenido del archivo o los n bytes leídos.
  • readline(): lee una línea del archivo y avanza el cursor.
  • readlines(): lee todas las líneas del archivo y devuelve una lista.

Escritura de Archivos

  • write("string"): escribe una línea en el archivo con string.
  • writelines(["string1", "string2"]): escribe todas las líneas en el archivo.

Mover el cursor

  • seek(desplazamiento, origen=0): Para escribir desde una posición determinada.
    • 0: desde el principio del archivo (default).
    • 1: desde la posición actual.
    • 2: desde el final.
  • tell(): devuelve en bytes la posición del cursor desde el principio del archivo.
desplazamiento puede ser positivo o negativo.

Tiempo

  • time(): segundos desde epoch.
  • gmtime(): tiempo en UTC (GMT).
  • localtime(): fecha y hora local (tupla).
  • sleep(segundos): duerme por segundos.
  • clock(): reloj de la CPU.
  • mktime(): convierte una tupla en tiempo time.
  • strftime(format, time=now): convierte a string.

from time import strftime

Sistema Operativo

os

  • listdir(ruta): devuelve la listado de directorio.
  • remove(ruta): elimina un archivo.
  • rename(ruta): cambia el nombre de un archivo.
  • chdir(ruta): cambia a un directorio de trabajo.
  • getcwd(): devuelve el directorio actual.
  • mkdir(ruta): crea un directorio.
  • rmdir(ruta): borra un directorio.
  • mkdirs(ruta): crea una jerarquía de directorios.
  • removedirs(ruta): elimina recursivamente.
  • renames(ruta): renombra recursivamente.

Sistema Operativo

os.path

  • basename(ruta): nombre base (archivo)
  • dirname(ruta): extrae ruta (directorio)
  • abspath(ruta): ruta absoluta
  • isfile(ruta), isdir(ruta): verifica el tipo
  • exists(ruta): verifica si existe la ruta
  • splitext(ruta): divide una ruta en (nombre, ext)
  • walk(path, visit, arg): recorre recursivamente los directorios, llamando a visit(arg, dirname, names)

Procesos del Sistema Operativo

os

  • system(comando): ejecuta un comando del sistema operativo.
  • fork():: inica un proceso hijo.
  • kill(pid, señal): envía una señal a un proceso.
  • waitpid(pid, options): espera un proceso.

RegEx.

Patrones más comunes

  • .: comodín de 1 carácter (menos \n).
  • |: or.
  • (): grupo.
  • []: clase.
  • [^]: negando.
  • \: escape de caracteres especiales.
  • +: una o más veces.
  • *: cero o más veces.
  • ?: cero o una vez.
  • {}: repetición.

Patrones más comunes

  • ^: anclaje al principio.
  • $: anclaje al final.
  • \d: [0-9]
  • \D: [^0-9]
  • \w: [a-zA-Z0-9_]
  • \W: [^a-zA-Z0-9_]
  • \s: [ \t\n\r\f\v]
  • \S: [^ \t\n\r\f\v]

Métodos de re

  • match(): busca que sea igual sino, None.
    • start
    • end
  • search(): busca el patrón.
    • start
    • end
  • group(): 0 es la cadena y el resto los grupos definidos.
  • groups(): devuelve una lista de los grupos.
  • findall(): devuelve una lista con las subcadenas que cumplen.
  • finditer(): devuelve un iterable para consultar uno a uno.
  • split(): devuelve una lista de subcadenas.
  • sub(): un patrón a substituir.

Sockets

La comunicación entre entidades de una red.

IP + Puerto + Protocolo = Socket

Conexión:

  • TCP: socket de flujo (socket.SOCK_STREAM)
  • UDP: socket de datagrama (socket.SOCK_DGRAM)

Familia:

  • Unix: basado en archivos (socket.AF_UNIX)
  • Red: basado en conexión (socket.AF_INET)

Usando Sockets

Creando un socket para el servidor


#Construimos el socket:
server = socket.socket()

#Definimos IP y Puerto:
server.bind(("localhost", 8000))

#Escuchamos X conexiones entrantes:
server.listen(5)

#Aceptamos una conexión:
client, (client_host, client_port) = server.accept()

Usando Sockets

Comunicándnos desde el servidor con el cliente


#Recibimos X bytes (recv = TCP, recvfrom = UDP):

#Mostramos lo recibido:

#Lo enviamos de vuelta:
client.send(mensaje)

Serialización y persistencia

pickle/cPickle convierte un objeto a una secuencia de bytes.

  • dumps(objeto, protocolo): serializa a un string.
  • dumps(objeto, archivo, protocolo): serializa a un archivo.
  • loads(cadena, protocolo): des-serializa un string.
  • load(archivo, protocolo): des-serializa desde un archivo.

Serialización y persistencia

shelve objeto similar a un diccionario persistente.

  • open(filename, flag='c', protocol=None, writeback=False): crea un diccionario persistente.
    • 'r': sólo lectura.
    • 'w': lectura y escritura.
    • 'c': creación, lectura y escritura.
    • 'n': nuevo.
  • sync(): sincronizar (writeback=True).
  • close()(): grabar y cerrar.
Protocolo:
0 - (default) texto.
1 - binario.
2 - binario más eficiente.
