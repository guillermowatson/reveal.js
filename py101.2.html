Tuplas y Listas

Listas

Es un conjunto de "cosas" que se guardan con un mismo nombre y son accesibles por posición.


>>> cosas = [10, 'lavandina', 3.14, "pan"]
>>> cosas
[10, 'lavandina', 3.14, "pan"]

Indexación

La primera posición siempre es cero.


>>> cosas[0]
10
>>> cosas[2]
3.14
>>> cosas[1]
lavandina
>>> cosas[5]
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list index out of range

Indexación

La última posición siempre es -1.


>>> cosas[-1]
pan
>>> cosas[-2]
3.14

Utilidades

Longitud

Si quiero saber el largo de una lista, puedo usar la función len.


>>> len(cosas)
4

Pertenencia

El operador in es un operador relacional y devuelve si el objeto de la izquierda está dentro del de la derecha.


>>> 'lavandina' in cosas
True
>>> 'aceite' in cosas
False

Recorriendo listas

Se puede recorrer una lista con un ciclo while.


>>> i = 0
>>> while i < len(cosas):
...     item = cosas[i]
...     print(item)
...     i = i + 1
10
lavandina
3.14
pan

Iteración

Pero mucho mejor es iterar con un ciclo for.


>>> for item in cosas:
...    print(item)
10
lavandina
3.14
pan

Trabajando con listas


>>> nuevas_cosas = cosas
>>> nuevas_cosas
[10, 'lavandina', 3.14, "pan"]
>>> nuevas_cosas.append('aceite')
>>> nuevas_cosas
[10, 'lavandina', 3.14, "pan", 'aceite']
>>> cosas
[10, 'lavandina', 3.14, "pan", 'aceite']

Mutabilidad

Las listas son objetos mutables. Se pasan por referencia.


>>> id(cosas)
140002257094984
>>> id(nuevas_cosas)
140002257094984

Mutabilidad

Los números y cadenas de texto son objetos inmutables. Se pasan por valor.


>>> a = 10
>>> b = 10
>>> id(a)
140002389619968
>>> id(b)
140002389619968
>>> b = 20
>>> id(b)
140002389620288
>>> a, b
(10, 20)

Copiando listas

Puedo usar slicing.


>>> cosas[2:]
[3.14, 'pan', 'aceite']
>>> cosas[:2]
[10, 'lavandina']
>>> cosas[1:4]
['lavandina', 3.14, 'pan']
>>> cosas[:]
[10, 'lavandina', 3.14, 'pan', 'aceite']

Copiando listas


>>> nuevas_cosas = cosas[:]
>>> id(cosas)
140002257094984
>>> id(nuevas_cosas)
140002254606472

Las cadenas se comportan como tuplas.

Diccionarios

Diccionarios

Son objetos mutables que guardan conjuntos de clave y valor.

Son indexables por las claves, no por números.


>>> eng_a_spa = {}
>>> eng_a_spa['one'] = 'uno'
>>> eng_a_spa['cat'] = 'gato'
>>> eng_a_spa
{'cat': 'gato', 'one': 'uno'}

Iterando sobre diccionarios


>>> for clave, valor in eng_a_spa.items():
...    print('{}: {}'.format(clave, valor))
cat: gato
one: uno

Ejemplos


>>> eng_a_spa['dog'] = 'perrito'
>>> eng_a_spa
{'cat': 'gato', 'one': 'uno', 'dog': 'perrito'}
>>> eng_a_spa['dog'] = 'perro'
{'cat': 'gato', 'one': 'uno', 'dog': 'perro'}

Módulos y paquetes

Módulos

Un archivo que contiene código Python, es un módulo.

Para entender mejor podemos consultar el Tutorial de Python

saludar.py


def saludar_mundo():
	print('Hola mundo')
						

>>> import saludar
>>> saludar.saludar_mundo()
Hola Mundo

Ejecutable


#!/usr/bin/env python3

def saludar_mundo():
	print('Hola mundo')

if __name__ == '__main__':
	saludar_mundo()
