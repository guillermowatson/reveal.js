¿Para quién es este curso?

Destinatarios

  • Programadores Python que quieren desarrollar en Web.
  • Estudiantes intermedios de Carreras de Informática.
  • Desarrolladores web que quieran aprender Python.
  • Todos aquellos con conocimientos básicos de programación y ganas de aprender a programar aplicaciones web en Python.

¿Qué tengo que saber para hacer curso?

Es necesario

  • Conocer la sintáxis y los diferentes tipos de datos de Python.
  • Entender qué es un paquete, un módulo, una función de Python.
  • Conocer la Programación Orientada a Objetos.
  • Manejar comandos de consola.
  • Tener muchas ganas de aprender, sin miedo a equivocarse.

Es recomendable

  • Saber utilizar Git.
  • Estar familiarizado con HTML, CSS y JavaScript.
  • Entender el funcionamiento de una Base de Datos.

¿Qué vamos a aprender durante el curso?

Objetivos

  • Entender la arquitectura de una aplicación web.
  • Desarrollar el código fuente de la aplicación.
  • Escribir la auto-documentación del proyecto.
  • Programar los tests automáticos.
  • Aprender a poner en producción nuestra aplicación!
